# 💡 LIGHTSON - Daily Expenses, Income & Investment Tracker

📱 **LIGHTSON** is a real-time personal finance tracking mobile app that helps users manage their **daily expenses**, **income sources**, and **investment plans** — all in one place. Built as a **full-stack project for LIGHTS INC**, the app provides data visualization with intuitive **pie charts** and real-time sync powered by the **MERN stack**.

---

## ✨ Features

- ✅ Daily Expense Tracking — Add, view, and manage your day-to-day spending.
- 💰 Income & Investment Management — Track multiple income sources and active investments.
- 📊 Data Visualization with Pie Charts — Instantly understand your finances with visual breakdowns.
- 🔄 Real-time Data Sync — All financial data is synced securely and instantly.
- 🔒 Authentication & Security — Login system with JWT to keep your data safe.
- 📱 Mobile-Optimized UI — Designed with React Native for smooth cross-platform performance.

---

## 🧠 Tech Stack

- **Frontend:** React Native (Expo)
- **Backend:** Node.js + Express
- **Database:** MongoDB Atlas
- **Visualization:** React Native Pie Chart
- 
### Prerequisites
- Node.js
- Expo CLI
- MongoDB (local or Atlas)
- Git
